*************
Globals
*************

Variables
---------

..  macro:: aerospike

    The :macro:`aerospike` variable is a global instance of the :type:`Aerospike` type.

Logging Functions
-----------------

All the logging functions accept a format string as the first parameter and a variable argument list.

..  function:: trace(s, ...)

    Log a trace message.

..  function:: debug(s, ...)

    Log a debug message.

..  function:: info(s, ...)

    Log an info message.

..  function:: warn(s, ...)

    Log a warning message.
